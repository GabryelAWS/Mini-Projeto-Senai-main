CSS External

Um CSS externo como o própio nome sugere, está fora do nosso documento! Analise ai o código fonte da pagina e veja o que foi feito na linha 8, dentro da área da área <head>.

As Principais vantagens no uso desse método são:

  • Como o arquivo esta externo, o seu contéudo (HTML)fica separado do estilo(CSS3). Isso Deixa tudo mais organiado
  • Um Mesmo arquivo de estilo pode ser usado em varias paginas do meu site. Eu preparei uma 💙 segunda pagina aqui 👈 que usa o mesmo estilo.
