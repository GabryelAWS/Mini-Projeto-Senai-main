CSS Internal

Um estilo interno é aquele que fica configurado dentro da área <head> do seu documento

A grande DesvantagemDessa técnica em relação ao modelo Inline é que podemos escrever uma configuração unica para todos os elemntos da mesma familia. Por exm todos os paragrafos desse documento estão com a mesma configuração de cor e eu só escrevi isso uma vez.

A grande DesvantagemDessa técnica em relação ao uso de um CSS Externo(Veja no proximo) é que o estilo não pode ser reaproveitado por outras paginas.

IMPORTANTE: Voce pode misturar CSS inline e CSS internal em um mesmo documento sem problemas! olha só o codigo fonte dessa pagina no seu navegador
